Ferramentas CASA

Ferramenta CASE é um produto baseado em computador que tem o propósito de ajudar os engenheiros de software em atividades relacionadas ao processo de desenvolvimento de software

Ela ajudar em:

  • ciração de Modelos de Entidades Relacionamento
  • Criação de Modelos Relacionais
  • Disponibilizar funcionalidades de criação de scripts de geração de banco de dados para tecnologias SGBDs
  • Possibilita a engenharia reversa

Algumas das ferramentas CASA são:

  • DBDesigner
  • ERWIN
  • DB-MAIN
  • IBM Rational
  • MySQLWorkbench

MySQULWorkbench

Iremos utilizar o MySQL por não possuir custo, por ser multiplataforma e ser amplamente utilizado. Ela também é uma ferramenta unificada para arquitetos de Banco de dados, desenvolvedores e DBAs.

Outro ponto é que ela fornece modelagem de dados, engenharia reversa, desenvolvimento de SQL e ferramentas de administração abrangentes para configuração de servidor, administração de usuário, backup e outros.
