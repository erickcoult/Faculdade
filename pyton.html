VARIÁVEIS E TIPOS BÁSICOS DE DADOS EM PYTHON

Variáveis são espaços alocados na memória RAM para guardar valores temporariamente. Em Python, esses espaços não precisam ser tipados, ou seja, a variável pode ser alocada sem especificar o tipo de dado que ela aguardará. O interpretador Python é capaz de determinar o tipo de dado da variável com base no seu valor, ou seja, as variáveis são tipadas dinamicamente nessa linguagem.

Possuimos algunas formas para se imprimir textos em PY, sendo eles:

# Modo 1 - usando formatadores de caracteres (igual na linguagem C) para imprimir variável e texto print("Olá %s, bem vindo a disciplina de programação. Parabéns pelo seu primeiro hello world" % (nome)) Olá João, bem vindo a disciplinade programação. Parabénspelo seu primeiro helloo world

Por format()

# Modo 2 - usando a função format() para imprimir variável e texto print("Olá {}, bem vindo a disciplina de programação. Parabéns pelo seu primeiro hello world".format(nome))

Por strings

# Modo 3 - usando strings formatadas print(f"Olá {nome}, bem vindo a disciplina de programação. Parabéns pelo seu primeiro hello world")

LEMBRE: Para fazermos comentário em Python, utilizamos a #

Operações matemática em Python

Igualmente em matemática, possuimos os sinais matemáticos e as operações.

  • x + y =>Soma de x e y
  • x - y =>Diferença de x e y
  • x * y =>Produto de x e y
  • x / y =>Quociente de x e y
  • x // y =>Parte inteira do quociente de x e y
  • x % y =>Resto de x / y
  • abs(x) =>Valor absoluto de x
  • pow(x, y) => x elevado a y
  • x ** y => x elevado a y

Sabendo dessas coisas, agora podemos montar a nossa expressão númerica de segundo grau. Primeiro, devemos definir o valor de cada letra e, inserirmos um input para o usuário colocar o valor que desejar.

a = 2 b = 0.5 c = 1 x = input("Digite o valor de x: ") x = float(x) y = a * x ** 2 + b * x + c print(f"O resultado de y para x = {x} é {y}.")

Ao usar a função input(), ela retorna uma string, independente do que o usuário digitou, sempre será string. Portanto, para corrigir o erro, precisamos converter o resultado da entrada em um tipo numérico. Para isso, utilizamos o "float()" (como apresentado acima), que permitirá o sistema dar continuidade a expressão. Caso contrário, aparecerá o ero " TypeError ". E, para verificarmos o type de cada elemento, utilizamos o código print(type(x)) para cada elemento.

ESTRUTURAS CONDICIONAIS EM PYTHON: IF, ELIF, ELSE

If

Assim como diz no nome, if significa "se" e else "senão". Eles tem como função de dar dois caminhos para o código. Caso o valor for verdadeiro (if) seguirá um resultado. Agora, se for falso (else), seguirá outro.

Após inserirmos esses comandos, temos de colocar as funções que serão executadas.

a = 5 b = 10 if a < b: print("a é menor do que b") r = a + b print(r)

Assim, se A for menor que B, o programa irá printar na tela " a é menor do que b" e logo em seguida fará o calculo de r, printando na tela após.

Else

Agora, caso desejamos acrescentar o outro caminho, utilizamos o Else. Ele fará o trabalho de mostrar outro resultado

a = input('valor 1: ') b = input('valor 2: ') if a < b: print('B é maior que A') else: print("A é maior do que B")

Ou seja, se o resultado de A or maior que B, aparecerá a função que está em ELSE

Elif

Basicamente o elif funciona como um source do HTML. Ou seja, caso a opção 1 não esteja de acordo( ou nao exista), será apresentado a opção 2; Caso a 2 n]ao esteja funcionando, será apresentada a opção 3 e apor assim vai.

codigo_compra = 5111 if codigo_compra == 5222: print("Compra à vista.") elif codigo_compra == 5333: print("Compra à prazo no boleto.") elif codigo_compra == 5444: print("Compra à prazo no cartão.") else: print("Código não cadastrado")

Assim, caso o código da compra for 5333, sistema irá printar "compra à prazo no boleto". Do contraio, irá ignorar o "compra à vista" e o "compra à prazo no cartão". E se não houver nenhum dentre as opções, o sistema printará o else

ESTRUTURAS LÓGICAS EM PYTHON: AND, OR, NOT

Além dos operadores relacionais, podemos usar os operadores booleanos para construir estruturas de decisões mais complexas.

  • Operador booleano and: Esse operador faz a operação lógica E, ou seja, dada a expressão (a and b), o resultado será True, somente quando os dois argumentos forem verdadeiros.
  • Operador booleano or: Esse operador faz a operação lógica OU, ou seja, dada a expressão (a or b), o resultado será True, quando pelo menos um dos argumentos for verdadeiro.
  • Operador booleano not: Esse operador faz a operação lógica NOT, ou seja, dada a expressão (not a), ele irá inverter o valor do argumento. Portanto, se o argumento for verdadeiro, a operação o transformará em falso e vice-versa.

Assim como as operações matemáticas possuem ordem de precedência, as operações booleanas também têm. Essa prioridade obedece à seguinte ordem: not primeiro, and em seguida e or por último

ESTRUTURAS DE REPETIÇÃO EM PYTHON: WHILE E FOR

A estrutura de repetição são dadas por while (enquanto) e por for(para). Eles tem como objetivo de repetir a execução do código. Assim, podemos programar para o programa fazer a pergunta X, dar-nos a resposta e, logo em seguida, fazer a pergunta X novamente, repetindo o processo.

LEMBRE: todos os blocos de comandos em Python são controlados pela indentação.

Usamos o comando "for" seguido da variável de controle "c", na sequência o comando "in" e, por fim, a sequência sobre a qual a estrutura deve iterar

nome = "Guido" for c in nome: print(c)

Assim, o resultado seguira cada letra de Guido em cada linha

Também podemos utilizar, junto com o "for", a função enumerate(). Ela tem como objetivo de enumerar a variável e de mostrar a posição.

nome = "Guido" for i, c in enumerate(nome): print(f"Posição = {i}, valor = {c}")
