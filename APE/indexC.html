Bibliotecas

As primeiras linhas de programação são definidas pelas bibliotecas, sendo algumas delas:

  • stdio: funções de entrada e saída
  • stdlib: transforma string em números
  • string: manipulação de string
  • math: operações matemáticas

Operador Lógico

Assim como nas outras linguagens, temos os operadores lógicos de C

  • && --> Lógico E- conjunção
  • || --> Lógico OU - disjunção
  • ! --> Lógico Não - negação

Comando de Saída de dados

As informações, mensgens e conteúdo das variáveis são enviadas para a tela do usuário

  • printf("expressão de controle", lista de argumentos);

Com ela, temos uma lista de argumentos, sendo:

  • %c = Permite a escrita de apenas um caractere.
  • %d = Permite a escrita de números inteiros decimais.
  • %e = Realiza-se a escrita de números em notação científica.
  • %f = É feita a escrita de números reais (ponto flutuante/float)
  • %s = Efetua a escrita de uma série de caracteres

Como exemplo temos printf("o valor encontrado foi %d", valor1);
